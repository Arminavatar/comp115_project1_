# Moonlit Night

A simple animated night scene made with Python's `turtle` module.

This project creates a peaceful moonlit landscape with:

- a glowing moon
- twinkling stars
- shooting stars
- pine trees
- a house with lit windows
- a pond
- chimney smoke animation

## Preview

![Moonlit Night Demo](progect%201.gif)

## Features

- Animated star effects
- Random shooting stars
- Smooth night sky gradient
- House and landscape drawing
- Smoke animation from the chimney
- Built using only Python standard libraries

## Built With

- Python
- Turtle
- Random
- Math
