This project is a simple night landscape created using Python Turtle graphics.
It draws a starry sky, a moon, grass, and a small house with a roof and door.


Rendering is optimized using:
turtle.tracer(0, 0)
This disables real-time drawing animation and updates the screen at once for better performance when rendering 1000 stars.



The goal of this project was to practice:

Functions
Loops
Random values
Basic geometry
Turtle graphics
Clean code structure

The program generates:
1000 random stars
A grey moon
Green grass
A house with:
Square base
Roof
Side structure
Door

Key Concepts Used:
for loops for repeated drawing
Random positioning for stars
begin_fill() and end_fill() for shapes
Functions to avoid repetition
Coordinate positioning with goto()

What I Learned"

How to structure a project using functions
How Turtle positioning works
How to combine shapes to build objects
How to use loops to create patterns
How to organize and document a project for GitHub
